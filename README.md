# Geografia Industrial: Coreia do Sul 🇰🇷

Este é um projeto web desenvolvido para o trabalho de **Geografia Industrial** (AC1 e PO1), com foco na potência econômica e tecnológica da **Coreia do Sul**.

## 🚀 Tecnologias Integradas
- **React.js** (Vite + TypeScript)
- **Framer Motion** para animações baseadas no scroll
- **Chart.js** (`react-chartjs-2`) para gráficos de exportação e importação
- **Lucide Icons**

## 📐 Estrutura do Site
O site foi construído num formato moderno de *Slides* de apresentação ('scroll-telling'). As seções implementadas seguem estritamente as regras exigidas no trabalho:
1. **Hero**: Introdução visual ao país.
2. **Identidade**: Capital (Seul), Bandeira e Emblema em formato limpo.
3. **Dados**: Área territorial, população total e PIB (Micro/Macroeconomia).
4. **Exportações**: Gráfico de produtos/parceiros (destaque para China, EUA, Semicondutores).
5. **Importações**: Gráfico dos países fornecedores de recursos (petróleo bruto - Arábia Saudita).
6. **Indústria**: Principais produtos industriais e curiosidade sobre semicondutores.
7. **Conclusão**: Explicação breve sobre o motivo da escolha e marcas representantes na página final.

## 🛠️ Como rodar localmente

Clone este repositório e rode os seguintes comandos na pasta `site-coreia-v2`:

```bash
# Instauração das dependências
npm install

# Rodar o servidor de desenvolvimento
npm run dev
```

Pronto! Acesse de volta no seu navegador a URL aberta (geralmente `http://localhost:5173`).
